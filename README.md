# Fetcho-Tool
A lightweight Python-based alternative to curl, optimized for quick domain fetching and clean terminal output.

## Run the following commands to start fetching:
```BASH (LINUX)
bash <(curl -s https://raw.githubusercontent.com/featherrz/Fetcho-Tool/main/setup.sh)
```

## Error Codes Meaning:
404: Page not Found (Error)

202: Page Found (Successful)

416: Invaild Domain (Error)
